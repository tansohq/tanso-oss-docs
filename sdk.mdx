The Tanso Core repository includes a typed, dependency-free client at
`sdks/typescript`. The runnable Next.js example consumes it as the
`@tansohq/sdk` workspace package.

```bash
git clone https://github.com/tansohq/tanso-oss.git
cd tanso-oss
npm install
npm run build --workspace @tansohq/sdk
```

```typescript
import { TansoClient } from "@tansohq/sdk";

const tanso = new TansoClient({
  apiKey: process.env.TANSO_API_KEY!,
  baseUrl: process.env.TANSO_BASE_URL ?? "http://localhost:8080",
});
```

<Warning>
Create the client only in server-side code. A Tanso API key authenticates the
whole account and must never enter a browser bundle.
</Warning>

## The integration loop

```typescript
const requestId = crypto.randomUUID();

// 1. Check before incurring provider cost.
const decision = await tanso.evaluateEntitlement({
  customerReferenceId: "demo-user",
  featureKey: "ai.chat",
  usage: {
    eventName: "ai.chat.generate",
    usageUnits: 1,
  },
  context: {
    idempotencyKey: `check-${requestId}`,
    flowId: requestId,
  },
});

if (!decision.allowed) {
  return deny(decision.meta?.reason?.description);
}

// 2. Perform the billable work.
const result = await runModel();

// 3. Record usage, unit economics, and one credit deduction.
await tanso.ingestEvent(
  {
    customerReferenceId: "demo-user",
    featureKey: "ai.chat",
    eventName: "ai.chat.completed",
    flowId: requestId,
    usageUnits: 1,
    costAmount: result.providerCost,
    revenueAmount: 0.02,
  },
  { idempotencyKey: `usage-${requestId}` },
);
```

## Surface

| Method | Client API operation |
| ------ | -------------------- |
| `createCustomer` | Create a customer |
| `getCustomer` | Get a customer by `customerReferenceId` |
| `checkEntitlement` | Read the current allow/deny decision |
| `evaluateEntitlement` | Simulate requested usage before work |
| `ingestEvent` | Record idempotent usage, cost, and revenue |
| `listCreditPools` | List a customer's balances |
| `getCreditPool` | Read one credit pool |

## Errors are typed

```typescript
import { TansoApiError } from "@tansohq/sdk";

try {
  await tanso.ingestEvent(event, { idempotencyKey: eventId });
} catch (error) {
  if (error instanceof TansoApiError && error.status === 409) {
    // A duplicate idempotency key or depleted hard-limit pool.
  }
}
```

`TansoApiError` exposes `status`, optional `detail`, and the parsed API
`response`.

<Note>
`@tansohq/sdk` currently ships in the GitHub repository, not the public npm
registry. Use the workspace package directly or pack it for your application.
The [included Next.js
example](https://github.com/tansohq/tanso-oss/tree/main/examples/nextjs-ai-credits)
is the canonical integration.
</Note>

List responses return `{ items, pagination }`. The client unwraps Tanso's
top-level `{ success, data, error }` envelope and throws on non-success
responses.
